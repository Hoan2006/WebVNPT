Trần Khải Hoàn - HƯỚNG DẪN CHẠY PROJECT
Bước 1: Clone project về từ GitHub
Bước 2: Mở project bằng VS Code có tải sẵn node js phiên bản v22.14.0
Bước 3: Mở terminal và chạy lệnh npm install
Bước 4: Chạy lệnh cd .\vnpt-landing-page\
Bước 5: Chạy lệnh npm install
Bước 6 : Chạy frontend bằng lệnh npm run dev
Bước 7: Mở thêm terminal và chạy backend bằng lệnh node server.js
( Nếu chạy node server.js báo không cho truy cập thì tự import data trong thư mục data e để trên GitHub sẵn luôn rồi có thể đổi chút xíu trong file .env hoặc server nữa, chạy được thì quá ngon tại em mới đổi atlas qua IP ai cũng coi được rồi)
